# mp-commons

Python shared kernel and commons library for microservices.

## Features

- **Kernel**: Commands, queries, events, CQRS dispatch, domain model primitives
- **Application**: Sagas, outbox, inbox, workflows, GDPR, masking
- **Config**: Settings validation, secrets loaders, dynamic hot-reload, feature flags
- **Observability**: Structured logging, metrics, tracing, health checks, SLO tracking
- **Security**: Encryption, API keys, JWT, mTLS, secret rotation
- **Resilience**: Circuit breaker, retry, rate limiting, bulkhead, fallback
- **Testing**: Fakes, fixtures, snapshot tests, approval tests, load testing

## Installation

```bash
pip install mp-commons
```

See [Quickstart](guides/quickstart.md) for a 5-minute setup guide.
