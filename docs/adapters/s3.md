# S3 / MinIO Object Storage Adapter Runbook

## Installation

```bash
pip install 'mp-commons[s3]'      # or: pip install aiobotocore
```

## Required Environment Variables

| Variable | Example | Description |
|---|---|---|
| `AWS_ACCESS_KEY_ID` | `AKIA...` | AWS (or MinIO) access key |
| `AWS_SECRET_ACCESS_KEY` | `...` | AWS (or MinIO) secret key |
| `AWS_DEFAULT_REGION` | `us-east-1` | AWS region |
| `S3_ENDPOINT_URL` | `http://localhost:9000` | *(MinIO only)* endpoint override |

## Basic Usage

```python
from mp_commons.adapters.s3 import S3ObjectStore

store = S3ObjectStore(bucket="my-bucket", region="us-east-1")
async with store:
    await store.put("uploads/file.pdf", content=pdf_bytes)
    data = await store.get("uploads/file.pdf")
    url = await store.presigned_url("uploads/file.pdf", expires_in=3600)
    await store.delete("uploads/file.pdf")
    keys = await store.list_keys(prefix="uploads/")
```

## MinIO (Local Development)

```python
store = S3ObjectStore(
    bucket="dev-bucket",
    endpoint_url="http://localhost:9000",
    aws_access_key_id="minioadmin",
    aws_secret_access_key="minioadmin",
)
```

## Common Error Codes

| Error | Cause | Fix |
|---|---|---|
| `botocore.exceptions.ClientError: NoSuchBucket` | Bucket doesn't exist | Create bucket before first use |
| `botocore.exceptions.ClientError: AccessDenied` | IAM permission missing | Add `s3:PutObject`, `s3:GetObject` etc. to IAM policy |
| `KeyError` from `store.get()` | Object key not found | Check key spelling; use `list_keys()` to verify |
| `botocore.exceptions.EndpointResolutionError` | Wrong region | Match `region` to bucket's actual region |
| `botocore.exceptions.ConnectTimeoutError` | Network timeout | Check VPC/security group; enable VPC endpoint for S3 |

## Performance Tuning

- **Multipart upload**: For files > 100 MB, use multipart upload (handled automatically by `aiobotocore` with `transfer_config`).
- **Transfer acceleration**: Enable S3 Transfer Acceleration for cross-region uploads.
- **Connection pool**: Default boto3 pool is 10 connections; increase via `config=Config(max_pool_connections=50)`.
- **Presigned URLs**: Presigned URLs are generated locally (no network call) and are valid for `expires_in` seconds.
